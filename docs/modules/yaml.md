# Prelude YAML

## Overview

This module bundles [yaml-mode](https://github.com/yoshiki/yaml-mode) for
editing YAML files.

## Features

Even though `yaml-mode` doesn't derive from `prog-mode`, the module enables
several useful features:

- **whitespace-mode** for visualizing whitespace (important in
  a whitespace-sensitive format like YAML)
- **subword-mode** for CamelCase aware editing
- **Automatic whitespace cleanup** on save
