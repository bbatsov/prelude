# Prelude CoffeeScript

!!! Note

    This module builds on top of the shared [Programming](programming.md) module.

## CoffeeScript Mode

This module bundles [coffee-mode](https://github.com/defunkt/coffee-mode) for
editing CoffeeScript files.

Prelude configures the following defaults:

- **Tab width**: 2 spaces
- **Compile-on-save**: If a compiled `.js` file already exists
  for the current CoffeeScript file, it will be automatically
  updated on save
- **No header comment**: The `"Generated by CoffeeScript"`
  header is removed from compiled output
- **CamelCase** aware editing (via `subword-mode`)
